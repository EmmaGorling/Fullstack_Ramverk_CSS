Startsida

Beskrivning av momentet

I kursens första moment ska vi bekanta oss med ramverk för CSS.
I momentet ska två undersidor skapas, i dessa sidor ska det finnas information om ramverket som använts, hur det har gått, en tabell, samt ett formulär som är kopplat till tabellen. Dessa sidor ska sedan stylas med de CSS-ramverk som valts.
Jag har valt att använda Bootstrap och Tailwind till undersidorna, då dessa är de populäraste ramverken just nu.

Om startsidan

Istället för att styla denna sidan och även publiceringssidan med vanlig CSS valde jag använda Bulma, för att jag ville testa några fler ramverk.
Mer information om Bulma hittar du på deras webbplats, bulma.io
Bulma känns som ett väldigt "litet" ramverk och jag kan tycka att det är väldigt svårt att hitta simpla saker i dokumentationen. Till exempel försökte jag få till att footern skulle hålla sig längst ner på sidan, trots lite innehåll, men jag kan inte hitta det någonstans och gick över till att skriva stylingen direkt i elementet för att få till detta. Även navigeringen var svår att få till utan att använda "style"-attributet i taggarna.
