Bootstrap

Om Bootstrap

Till stylingen för denna sida har ramverket Bootstrap använts. Det är det populäraste CSS-ramverket just nu som används för att "snabbt och enkelt" styla responsiva webbplatser. Med Bootstrap använder man fördefinierade klasser som läggs till elementen för att tillämpa stylingen där det även finns färdig styling för komponenter, till exempel knappar.
Vill du ta reda på mer information och och läsa dokumentation finns detta på deras webbplats, getbootstrap.com.

Att jobba med Bootstrap

Att ge sig på Bootstrap som första CSS-ramverk var inget enkel match. Under större delen av utvecklingen av denna sidan har jag funderat på om det verkligen är någon som använder det, då det känns alldeles för krångligt. Det är i det närmaste omöjligt att försöka förstå hur man ska få till en responsiv sida som ser bra ut på stora skärmar. Man blir även kanske låst till ramverket och jag kan förstå vad lärarna menar med att en sida lätt kan se lite "Bootstrapig ut". Det enda som jag kan känna är bra med Bootstrap är det faktiskt har färdig CSS för vissa komponenter, som till exemepel knappar.
