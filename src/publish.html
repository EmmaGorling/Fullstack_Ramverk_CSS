Publicering

Vanligtvis publicerar jag i princip allt som hör till skolan via Netlify, jag tycker det är väldigt smidigt och jag har lärt mig skapligt hur det fungerar.
Men för det här momentet tänkte jag att skulle prova något annat. Först testade jag GitHub Pages, det verkade gå smidigt och bra att deploya, fram till att jag skulle besöka sidan. Då står det 404 och att det inte finns någon index.html-fil och strax därefter fick jag ett mail om att deployen inte fungerade. Jag försökte hitta vad det kunde bero på, men hittade inte anledningen.
Därefter testade jag Render, men av någon anledning fungerar inte det heller. Jag får bara upp deploy failed..
Och gissa vad som fungerade direkt, utan några kosntigheter eller knussel? Jo självklart Netlify! Så jag gjorde en deploy därifrån men kände att jag ville ändå försöka publicera det via en annan plattform. Så jag testade även Vercel, som jag aldrig har testat innan, och fick till det där med.

Netlify

Netlify är en populär plattform som används för att bygga och publicera webbplatser och webbapplikationer. Med Netlify finns det möjlighet att koppla sitt versionshanteringssystem (t ex. GitHub eller GitLab) och därmed också göra deploys direkt från sitt repositrory. När man sedan gör en ny push till sitt repo kommer den publicerade sidan automatiskt uppdateras med den nya versionen. För små hobby-projekt kostar det ingenting att använda Netlify, men det finns även betalplaner för större prjekt. Netlify har stort stöd för olika programmeringsspråk och utvecklingsverktyg: HTML, CSS, JavaScript, Node.js TypeScript, Python, PHP med flera.

För att lägga till en ny webbplats med Netlify går man till "Add new site" från startsidan (när man är inloggad). I den menyn kan man välja hur webbplatsen ska skapas, för att använda till exempel GitHub väljs "Import an existing project". Efter inloggning till git-plattformen väljer man det repo som ska publiceras ohc därefter ska eventuella inställningar läggas till. Sedan är det bara att trycka på Deploy och vänta på att webbplatsen skapas och går online.

Vercel

Vercel är en plattform för hosting och utveckling av webbapplikationer, som är särskilt optimerad för frontend-ramverk.
